# Rodízio de exercícios

Rotação simples, um item por disparo (ordem sugerida, pode variar):

1. Alongamento rápido — pescoço, ombros, costas (1-2 min).
2. Caminhada/levantar — levantar e caminhar alguns minutos.
3. Respiração/postura — respiração consciente + correção de postura.
4. Flexão de braço — algumas repetições, ritmo confortável.
5. Prancha — 20-40s, sem forçar.
6. Agachamento — algumas repetições, ritmo confortável.
